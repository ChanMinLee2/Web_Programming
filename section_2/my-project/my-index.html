HTML & CSS Basics Summary

HTML Summary

    HTML (HyperText Markup Language) is used to define our page content, structure and meaning. you don't use it or styling purposes. Use CSS for that instead!

  1. HTML uses "element" to describe (anotate) content
  2. HTML elements typically have an openning tag, content and then a closing tag
  3. You can also have void(empty) elements like images
  4. You can also configure elements but you'll gain experience over time, no warries.
  5. HTML uses "element" to describe (anotate) content

    6. Learn more about all available HTML elements on the MDN HTML element reference

CSS summary

    CSS (Cascading Style Sheets) is used for styling your page content.

  1. Styles are assigned via property-value pairs
  2. You can assign styles via the "styles" attribute
  3. To avoid code duplication, you typically use global styles(e.g. via the "style" element) instead
  4. Alternatively, you can work with external stylesheet files which you "link" to
  5. When working with CSS, concepts like "inheritance", "specificity", and the "box model" should be understood.

    6. Learn more about all available CSS properties and values on the MDN CSS property reference
